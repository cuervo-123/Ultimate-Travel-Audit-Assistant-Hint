# Travel Audit Assistant — Tech Blue+ (Fixed Help)

This build includes the **class-based Help overlay** (no `hidden` attribute), ensuring the modal opens/closes reliably on GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `assets/logo.svg`
- `assets/banner.svg`

## Deploy
1. Replace the files in your repository root with the contents of this package.
2. Commit & push to `main`.
3. (Optional) Keep a `.nojekyll` file in the repo root to avoid Jekyll interference.
4. Open your Pages URL: https://cuervo-123.github.io/Travel-Audit-Assistant/
